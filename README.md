# Sales Data Analysis Project

👩‍💻 Created by: Muskan Agrawal

## 📊 Project Overview
This is my first Data Analyst project using Excel. In this project, I analyzed sales data to find useful insights and trends.

## 🛠️ Tools Used
- Microsoft Excel

## 🔍 Work Done
- Data cleaning and formatting
- Created age groups (Teenager, Adult, Senior)
- Analyzed sales by channels (Amazon, Flipkart, Myntra, Meesho)
- Calculated total sales
- Created charts and dashboard

## 📈 Key Insights
- Amazon has the highest sales
- Most customers belong to the Adult age group
- Sales are higher in certain cities
